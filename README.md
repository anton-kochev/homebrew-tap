# Homebrew Tap

A collection of Homebrew formulas for developer tools by [anton-kochev](https://github.com/anton-kochev).

## Installation

First, tap this repository:

```bash
brew tap anton-kochev/tap
```

Then install any of the available formulas:

```bash
brew install <formula-name>
```

## Available Formulas

### ai-commit

A Rust-based AI commit message generator that helps you write meaningful commit messages automatically.

**Installation:**
```bash
brew install ai-commit
```

**Details:**
- **Homepage:** [github.com/anton-kochev/ai-commit](https://github.com/anton-kochev/ai-commit)
- **Supported Platforms:**
  - macOS (Intel & Apple Silicon)
  - Linux (x86_64)

### service-bus-inspector

Terminal-based UI for inspecting and monitoring Azure Service Bus queues.

**Installation:**
```bash
brew install service-bus-inspector
```

**Details:**
- **License:** MIT
- **Homepage:** [github.com/anton-kochev/service-bus-inspector](https://github.com/anton-kochev/service-bus-inspector)
- **Supported Platforms:**
  - macOS (Intel & Apple Silicon)
  - Linux (x86_64)

## Updating

To update the formulas to their latest versions:

```bash
brew update
brew upgrade <formula-name>
```

## Uninstallation

To remove a formula:

```bash
brew uninstall <formula-name>
```

To untap this repository:

```bash
brew untap anton-kochev/tap
```
